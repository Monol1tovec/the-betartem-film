# THE BETARTEM — Официальный сайт фильма

Современный кинематографичный сайт для полнометражного фильма THE BETARTEM.

## 🎬 Особенности

- **Современный дизайн** — тёмная кинематографичная тема с красными акцентами
- **Полностью адаптивный** — отлично выглядит на ПК и мобильных устройствах
- **Плавные анимации** — fade-in эффекты при прокрутке
- **Эффект частиц** — атмосферный фоновый эффект на Canvas
- **Встроенный видеоплеер** — Google Drive интеграция для просмотра фильма
- **Кинематографичное свечение** — анимированный glow-эффект для логотипа

## 🚀 Быстрый старт

### Установка зависимостей

```bash
npm install
```

### Запуск dev-сервера

```bash
npm run dev
```

Сайт будет доступен по адресу: `http://localhost:5173`

### Сборка для продакшена

```bash
npm run build
```

## 📦 Деплой на Vercel

### Способ 1: Через Vercel CLI

1. Установите Vercel CLI:
```bash
npm install -g vercel
```

2. Деплой:
```bash
cd the-betartem-film
vercel
```

### Способ 2: Через GitHub + Vercel Dashboard

1. Создайте репозиторий на GitHub
2. Запушьте код:
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin <your-repo-url>
git push -u origin main
```

3. Перейдите на [vercel.com](https://vercel.com)
4. Нажмите "Import Project"
5. Выберите ваш репозиторий
6. Vercel автоматически определит Vite и настроит сборку

### Способ 3: Через Vercel Dashboard (прямой деплой)

1. Перейдите на [vercel.com](https://vercel.com)
2. Нажмите "Add New" → "Project"
3. Выберите "Import Git Repository" или загрузите папку напрямую
4. Vercel автоматически развернёт проект

## 🛠️ Технологии

- **React 19** — современный UI фреймворк
- **Vite 8** — быстрый сборщик
- **CSS3** — кастомные стили и анимации
- **Canvas API** — эффект частиц
